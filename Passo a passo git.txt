INSTALAR O GIT:
depois de instalar no cmd
git --version
configurando git
git config --global user.name "digite seu nome"
git config --global user.email "alvaroxramos@gmail.com"
git config --global core.editor vs code

comandos basicos:
para iniciar : git init
verificar arquivos nao enviados : git status
enviar um arquivo: git add nome arq
enviar muitos arquivos: git add -A (git commit -am "")

para executar um commit : git commit -m "add comentario"
exibir log de atividades : git log // vevr tudo que ja fiz

exibir o branch : git branch  // partição que voce criar (nao essencial)

para voltar modificações: copiara codigo do commit 
git reset --soft  --mixed --hard

criar um branch: git brach "nome do brach"
mudar de branch: git checkout "nome do branch"

git remote add origin https://github.com/AndreJacob/Esp8266WebServer.git // adicionar url
git push -u origin master
